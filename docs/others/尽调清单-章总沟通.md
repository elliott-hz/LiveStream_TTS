# 数字人直播业务尽调清单

> **文档版本：** v1.0  
> **日期：** 2026-07-19  
> **用途：** 第二次沟通用 — 搞清楚现状才能出方案  
> **保密级别：** 商业机密  
> **给章总看吗：** ❌ 不要给。这是你自己的笔记。

---

## 一、商汤合作细节（最重要）

```
□ 合作模式？
   → SaaS 平台按用量付费？年度 license？GMV 分成？其他？

□ 合同周期？
   → 签了多久？什么时候到期？
   → 提前解约有什么惩罚？
   → 有没有排他条款？最低消费？

□ 费用
   → 月费 / 年费大概什么量级？（不需要精确数字）
   → 计费维度？（路数 × 时长？客户数？GMV 比例？）

□ 技术边界
   → 商汤提供了哪些模块？
     □ 数字人克隆
     □ TTS 语音合成
     □ 渲染 & 口型同步
     □ RTMP 推流
     □ 互动问答 / 弹幕回复
     □ 商家管理后台
     □ 数据统计 & 看板
     □ 其他：_______

   → 你们自己做了哪些？（哪怕只是写了个脚本也算）

□ 数据归属
   → 数字人形象资产归谁？能导出吗？什么格式？
   → 音色数据归谁？能导出吗？
   → 商家上传的商品/脚本/FAQ 数据在谁手里？能导出吗？
   → 直播历史数据（弹幕/回复/GMV）能导出吗？
   → 你们有数据库权限吗？还是纯黑盒？

□ API 开放性
   → 商汤有没有开放 API？
   → 你们能不能自己调用？（比如自己做互动、只调用渲染）
   → 还是必须走他们的全套平台？
```

## 二、业务核心指标（量级即可）

```
□ 客户
   → 1000 个客户中：付费比例？活跃开播比例？
   → 客单价范围？（月费多少到多少）
   → 前 20% 客户贡献多少收入？
   → 客户留存率大概多少？

□ 直播规模
   → 日均同时在线直播间数？⭐ 最重要
   → 峰值并发？
   → 每直播间平均开播时长？

□ GMV
   → 月 GMV 量级？（不方便说具体数字就说量级）
   → GMV 增长趋势？（稳定/增长/下降）

□ 收费模式
   → 你们向商家怎么收费？
     □ SaaS 月费订阅
     □ 按 GMV 抽佣
     □ 项目制定制
     □ 混合模式
```

## 四、团队 & 资源

```
□ 40 人构成
   → 销售多少人？
   → 运营/客服多少人？
   → 有没有技术背景的？（哪怕一个）

□ 如果自建技术团队
   → 你打算配几个人？
   → 招聘预算？
   → 你对技术团队的组织定位？（独立部门？跟我汇报？）

□ 技术预算
   → 心理预期的一次性投入？
   → 心理预期的月度固定支出？
   → 有没有已经留好的预算？

□ 时间线
   → 你期望多快看到东西？
   → 能接受的完整替换周期是多久？
```

---

## 五、客户 & 迁移

```
□ 客户结构
   → 大客户有多少？小客户有多少？
   → 大客户有定制需求吗？能接受产品切换吗？

□ 迁移容忍度
   → 存量客户能接受迁移过程中的波动吗？
   → 能不能分批迁移？（先新客后老客）

□ 商家原始素材
   → 克隆用的主播视频是商家自己上传的吗？
   → 你们手里有没有存一份？
   → 如果没有，能不能让商家重新录？
```
